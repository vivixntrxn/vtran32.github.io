# Pickleball Hub
Pickleball Hub is a beginner friendly website designed to introduce new players to the sport of pickleball.
It covers gameplay basics, equipment, places to play, and a feedback form for user suggestions.

This project was built using **HTML, CSS, and JavaScript**, with a focus on responsive design and accessible UI patterns.

---

## Features

### Responsive Layout
- Mobile first design with breakpoints at **600px** (tablet) and **1024px** (desktop).
- The navigation menu becomes a dropdown on mobile and expands horizontally on desktop.
- Learn section adapts from 1 column to 2 columns to 3 columns depending on screen size.

### Interactive Navigation
- Mobile/Tablet hamburger menu with open/close behavior.
- Automatic highlighting of the current page.

### "What You'll Learn Section
- 4 interactive cards:
  - **Gameplay Fundamentals**
  - **Equipment**
  - **Place to Play**
  - **Feedback**
- Each card is clickable and links to its corresponding page.
- Hover effects and animations for better user engagement.

### Video Section
- Embedded introductory pickleball video with description and credits.

### Footer
- Responsive footer with:
  - Pages list
  - Feedback link
  - Contact information
- Desktop version displays links horizontally with separators (`|`)

### Design System
- Custom color palette using CSS variables.
- Consistent typography using Georgia and Arial.
- Card components with hover scaling and rounded corners.
- Desktop shadow effects for the web page.
