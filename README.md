# vivixntrxn.github.io
This repository contains files for CIS300 Project Assignments.

Pickleball Hub is a beginner-friendly website designed to teach new players the fundamentals of pickleball.
It covers gameplay basics, equipment recommendations, and places to play. It also includes an introductory video
and a feedback form for user suggestions.

---

## Pages
- **Home (index.html)**: Overview of pickleball, what users will learn, and intro video
- **Gameplay Fundamentals**: Rules, scoring, terminology, and DUPR rankings
- **Equipment**: Paddle and ball recommendations. Buying guidance.
- **Places to Play**: Court finder and location resources.
- **Feedback**: User feedback form.

## Features
- Fully responsive design (mobile, tablet, and desktop)
- Custom color palette and typography
- Interactive navigation menu
- Image-based recommendation cards
- Structural tables for balls, paddles, and DUPR rankings
- Embedded video section
- Clean and beginner-friendly instructional content

## Technologies
- HTML5
- CSS (custom responsive grid system)
- JavaScript (navigation toggle & active link highlighting)
